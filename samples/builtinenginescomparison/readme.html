DMVCFramework Built-In Web Server Comparison

This project allows to compare the classic DMVCFramework built-in server based on INDY and the new engine, introduced in dmvcframework-4.0.0-oxygen, based on kernel module HTTP.sys.

There are 2 build configurations:

  • INDY: Which uses the INDY version
  • HTTPSYS: Which uses the new HTTP.sys version

Build project using the INDY configuration and then run apachebenchmark (instructions https://www.cedric-dumont.com/2017/02/01/install-apache-benchmarking-tool-ab-on-windows/) using the following commandline: 

c:\Tools\ab -n 5000 -c 100 http://localhost:8080/api

Note the value in "Requests per second"

The, stoip the server and rebuilt the project using the HTTPSYS configuration. Then restart the server and run the same command line.

c:\Tools\ab -n 5000 -c 100 http://localhost:8080/api

Note the value in "Requests per second".
